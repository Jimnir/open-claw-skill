# 面试辅导助手 🎯

> 面试题分析与回答生成器：拆解考察点、多角度回答、结合实战案例。

## 技能简介

**interview-tutor** 是一个面试准备辅助技能。无论你是准备技术面试、系统设计面试还是行为面试，这个技能都能帮你分析问题背后的考察点，提供多个角度的优质回答，并结合实际案例加深理解。

## 使用场景

| 场景 | 说明 |
|------|------|
| 💡 刷面试题 | 遇到不会的面试题，让助手分析并给出最优回答 |
| 📝 准备面试 | 系统性地复习和演练各类面试问题 |
| 🔬 深度理解 | 不仅知道答案，还要理解为什么这么考 |
| 🎭 模拟实战 | 从真实面试案例中学习回答技巧 |

## 触发方式

直接问面试题即可触发，例如：

- 「讲讲 TCP 三次握手」
- 「MySQL 索引的底层实现是什么」
- 「如何设计一个短链接系统」
- 「你遇到过最棘手的问题是什么」

## 工作流程

### 第一步：问题分析

助手会帮你拆解问题，找出面试官真正想问什么：

- **核心考察点**：这个问题在考什么知识点
- **隐藏考点**：有哪些容易被忽视的细节
- **难度分级**：初级 / 中级 / 高级

### 第二步：多角度回答

每个问题提供 3 个角度的回答：

| 版本 | 特点 | 适合谁 |
|------|------|--------|
| **回答 1（推荐）** | 结构清晰、深度适中、面试官最爱 | 大多数候选人 |
| **回答 2** | 深入细节、展示深度 | 高阶职位、想脱颖而出的 |
| **回答 3** | 简洁精炼、抓重点 | 限时回答、快速应答 |

### 第三步：实战案例

用 web_search 搜索真实面试案例或行业实践，给出：

- **实战实例 1**：真实场景中的应用
- **实战实例 2**：面试中可能遇到的追问或延伸

## 面试题类型覆盖

| 类型 | 示例问题 |
|------|---------|
| 📐 **计算机网络** | TCP/UDP 区别、HTTP 状态码、DNS 解析过程 |
| 🗄️ **数据库** | 索引原理、事务隔离级别、SQL 优化 |
| 💻 **操作系统** | 进程线程区别、内存管理、调度算法 |
| ⚙️ **系统设计** | 设计短链接、设计秒杀系统、设计消息队列 |
| 🏗️ **架构设计** | 微服务拆分、CAP 理论、分布式事务 |
| 👥 **行为面试** | 项目难点、团队冲突、失败经历 |
| 🔤 **语言基础** | Python/Go/Java 相关基础题 |

## 使用建议

1. **主动思考后再看答案**：先想一下你会怎么回答，再对比助手的分析，效果更好
2. **关注考察点**：比答案更重要的是理解面试官「为什么问这个问题」
3. **结合实际项目**：把实战案例和你自己的项目经验结合起来，面试时更有说服力
4. **追问问到底**：如果某个回答没看懂，让助手进一步展开讲解
